## 2025-05-22 - [Telegram Bot UX: Feedback and Clutter Reduction]
**Learning:** Providing immediate feedback via `send_chat_action` (typing...) and localized status messages improves perceived performance. Using `edit_text` to transition from a status message to the final content significantly reduces chat clutter, creating a more professional and pleasant interface.
**Action:** Always include immediate feedback and use message editing for status transitions in Telegram bots.
